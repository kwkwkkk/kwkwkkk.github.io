从小白开始搭建github blog

  1. 首先申请账号，这就不用说了……
  2. 连接github 必要的工具 git 安装也很简单就不写了

  3. node.js 这个js环境也很重要，安装也不用说
    到这里总结一下遇到的问题 安装node.js是默认会配置系统变量，但是用户变量没有配置，需要自己添加，变量配置也是艸鸡简单。。。不再赘述，node.js要先安装，不然要重新打开git bash才能用

  4. 在github新建Repository。。。格式是用户名,github.io，基本就是填下名字之类的就好了。

  5. Git连接github（这里是重点！！！）
    生成一个新的SSH。
    $ssh-keygen -t rsa -C “your e-mail”
    这里your email一定要填对！！！不然连接不上。。。我是第二次才连接成功的，第一次任性了。。。引以为戒。
    接下来就直接回车就好，不用管填什么东西。下面有提示在哪个目录生成了.ssh ，里面有两个文件：id_rsa , id_rsa.pub。id.rsa不用管，把id_rsa.pub添加进github，进入account-setting ，把id_rsa.pub的内容复制进去就可以了。
    在git bash 输入ssh -T git@github.com

这样就连接成功了

  1. 安装hexo
    在安装前先确认node配置好了吗。。。
    输入 node -v，出现版本号就好了

输入 npm install -g hexo-cli 等它一行行代码刷完就好了
新建hexo目录（还是重点）

进入目录后输入hexo init 又是等一堆东西刷完。。。
这里再来个错误提示：一定要给hexo一个文件夹。。。不然上面会报错的。。。别问我怎么知道的

还没完呢。。。接着输npm install
好了。。。运行一下hexo server 成功后http://localhost:4000/ 就能看到效果了
